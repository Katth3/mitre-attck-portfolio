# T1059.001 — PowerShell Suspicious Execution

**Tactic:** Execution  
**Technique:** T1059.001 — Command and Scripting Interpreter: PowerShell  
**Data source:** Sysmon Event ID 1 (Process Creation)  
**Threat actors:** APT29, Lazarus Group, FIN7  

---

## Query 1 — Encoded command detection

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=1
(Image="*\\powershell.exe" OR Image="*\\pwsh.exe")
(CommandLine="*-EncodedCommand*" OR CommandLine="*-enc *" OR CommandLine="*-ec *")
| table _time, ComputerName, User, Image, CommandLine, ParentImage, ParentCommandLine
| sort -_time
```

**What it detects:** PowerShell launched with encoded commands — base64 obfuscation
commonly used by malware to hide malicious scripts from defenders.

**Expected result in a clean environment:** Zero or very few results.
Any hit should be investigated.

---

## Query 2 — Suspicious PowerShell flags

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=1
(Image="*\\powershell.exe" OR Image="*\\pwsh.exe")
(CommandLine="*-NonInteractive*" OR CommandLine="*-WindowStyle Hidden*"
OR CommandLine="*-NoProfile*" OR CommandLine="*bypass*")
| table _time, ComputerName, User, CommandLine, ParentImage
| sort -_time
```

**What it detects:** PowerShell running silently — hidden window, no profile,
execution policy bypass. Common pattern in malware droppers and C2 stagers.

**Tuning notes:** Filter out known admin tools and deployment software
(SCCM, Intune, backup agents) by adding:
```splunk
NOT (ParentImage="*\\ccmexec.exe" OR ParentImage="*\\IntuneManagementExtension.exe")
```

---

## Query 3 — PowerShell spawned by Office applications

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=1
(Image="*\\powershell.exe" OR Image="*\\cmd.exe" OR Image="*\\wscript.exe")
(ParentImage="*\\WINWORD.EXE" OR ParentImage="*\\EXCEL.EXE"
OR ParentImage="*\\OUTLOOK.EXE" OR ParentImage="*\\MSPUB.EXE")
| table _time, ComputerName, User, Image, CommandLine, ParentImage
| sort -_time
```

**What it detects:** Office applications spawning shells — the classic indicator
of a malicious macro executing. High confidence alert.

**Expected result:** Zero results in a healthy environment.
Any hit is a critical finding — investigate immediately.

---

## False positives

| Source | Description | Mitigation |
|--------|-------------|------------|
| SCCM / Intune | Software deployment uses encoded PS | Whitelist by ParentImage |
| Backup agents | Some use hidden PowerShell windows | Whitelist by ComputerName |
| Admin scripts | Legitimate automation with -NonInteractive | Document and whitelist |
