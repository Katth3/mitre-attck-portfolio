# T1003.001 — LSASS Memory Access

**Tactic:** Credential Access  
**Technique:** T1003.001 — OS Credential Dumping: LSASS Memory  
**Data source:** Sysmon Event ID 10 (Process Access)  
**Threat actors:** APT29, Lazarus Group, Mimikatz users  
**Confidence:** High when filtered correctly

---

## Query 1 — LSASS access by suspicious processes

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=10
TargetImage="*\\lsass.exe"
NOT (SourceImage="*\\MsMpEng.exe"
OR SourceImage="*\\svchost.exe"
OR SourceImage="*\\wininit.exe"
OR SourceImage="*\\csrss.exe"
OR SourceImage="*\\wmiprvse.exe")
| table _time, ComputerName, SourceImage, TargetImage, GrantedAccess, CallTrace
| sort -_time
```

**What it detects:** Any process accessing LSASS memory that is not a known
legitimate system process. Mimikatz, ProcDump, and similar tools all trigger
Sysmon EID 10 when accessing LSASS.

---

## Query 2 — High-risk access rights to LSASS

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=10
TargetImage="*\\lsass.exe"
(GrantedAccess="0x1010" OR GrantedAccess="0x1038"
OR GrantedAccess="0x1410" OR GrantedAccess="0x143a"
OR GrantedAccess="0x1fffff")
| table _time, ComputerName, SourceImage, GrantedAccess, CallTrace, User
| sort -_time
```

**What it detects:** Specific memory access rights associated with credential
dumping tools. These access masks are documented in Mimikatz source code
and are highly suspicious when requested by non-system processes.

---

## Query 3 — LSASS access frequency baseline

```splunk
index=main sourcetype="XmlWinEventLog:Microsoft-Windows-Sysmon/Operational"
EventID=10
TargetImage="*\\lsass.exe"
| stats count by SourceImage, ComputerName
| sort -count
```

**What it detects:** Baseline view of all processes accessing LSASS.
Run this first to understand your environment before tuning Query 1.
Anything unexpected in the top results warrants investigation.

---

## False positives

| Source | Description | Mitigation |
|--------|-------------|------------|
| AV / EDR agents | Legitimate memory scanning | Whitelist by SourceImage path |
| Password managers | Some access LSASS for SSO | Verify and whitelist specific path |
| Windows Defender | MsMpEng.exe accesses LSASS regularly | Already filtered in Query 1 |
